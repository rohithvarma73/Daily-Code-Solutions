# Day106 – Find N Unique Integers Sum up to Zero
**Solved – Easy**

---

### Problem

Given an integer `n`, return **any array containing `n` unique integers** such that they add up to `0`.

---

### Example 1

**Input:**
```python
n = 5
````

**Output:**

```python
[-7, -1, 1, 3, 4]
```

**Explanation:**
These arrays are also valid:

* `[-5, -1, 1, 2, 3]`
* `[-3, -1, 2, -2, 4]`

---

### Example 2

**Input:**

```python
n = 3
```

**Output:**

```python
[-1, 0, 1]
```

---

### Example 3

**Input:**

```python
n = 1
```

**Output:**

```python
[0]
```

---

### Constraints

* `1 <= n <= 1000`
